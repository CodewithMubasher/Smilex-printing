# Smilex Printing Shop

A modern, responsive website for Smilex Printing Shop built with Next.js 16 and Tailwind CSS v4.

## Features

- Responsive design (mobile-first)
- Services showcase with image cards
- About section with animated counters
- FAQ accordion
- Contact form with mailto integration
- WhatsApp integration with pre-filled message
- Google Maps embed for shop location
- Dark mode support

## Tech Stack

- **Framework:** Next.js 16.1
- **Styling:** Tailwind CSS v4
- **Animations:** Motion (motion/react)
- **UI Components:** shadcn/ui
- **Package Manager:** pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Build

```bash
pnpm build
```
